## 🚀 Für ein neues Projekt

### Schritt 1: Clone den Branch

```bash
git clone https://github.com/saturas89/remember-me.git my-new-project
cd my-new-project
git checkout new-project
```

### Schritt 2: Neuer Branch für dein Projekt

```bash
git checkout -b develop
```

### Schritt 3: Anpassen

```bash
# Ändere package.json name
# Ändere docs/PROJECT.md
# Schreibe neue README.md
```

### Schritt 4: Dokumentation schreiben

```
1. docs/requirements/ Anforderungen
2. docs/modules/ Module definieren
3. Code entwickeln
```

### Schritt 5: Pushen

```bash
git push -u origin develop
# Vercel deployt automatisch!
```
