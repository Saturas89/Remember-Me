# Code of Conduct

## Our Pledge

We want Storyhold to be a welcoming, respectful, and inclusive community.

Participants are expected to:

- Be respectful and constructive
- Show empathy and patience
- Respect different experiences and viewpoints
- Focus on collaboration
- Avoid harassment and personal attacks

## Unacceptable Behavior

Examples include:

- Harassment
- Discrimination
- Personal attacks
- Publishing private information
- Trolling or intentionally disruptive behavior

## Enforcement

Project maintainers may remove or moderate comments, discussions, pull requests, or contributions that violate this code of conduct.

## Scope

This Code of Conduct applies to:

- GitHub discussions
- Issues
- Pull requests
- Community interactions around Storyhold
