## 🎯 MoSCoW Priorisierung

### MUST (Muss – ohne diese geht es nicht)
- Frage-Flow mit Kategorien und spielerischer Führung
- Antworten lokal speichern (kein Datenverlust beim Schließen)
- Offline-Nutzung (PWA)
- Lebensarchiv ansehen (alle gespeicherten Antworten)

### SHOULD (Sollte – wichtig für gute UX)
- Fortschrittsanzeige pro Kategorie
- Antworten nachträglich bearbeiten
- Export als PDF oder druckbares Dokument
- Responsive Design (Mobile-first)

### COULD (Könnte – nettes Extra)
- Fotos zu Antworten hinzufügen
- Teilen-Link für Familienmitglieder generieren
- Push Notifications als Erinnerung
- Mehrsprachigkeit (DE / EN)

### WON'T (Wird nicht – bewusst ausgeschlossen)
- Eigene Server-Infrastruktur / Backend (kein eigenes Backend in v1)
- Social-Media-Funktionen
- Monetarisierung / Paywall
