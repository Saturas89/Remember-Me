# Modul: Workout Management

## 📋 Übersicht

Das Workout Management Modul verwaltet Trainingsplan und Übungen. Benutzer können Workouts erstellen, bearbeiten und organisieren mit vollständiger CRUD-Funktionalität.

- Workout CRUD Operationen

- Workout Templates & Vorlagen
- Übungs-Reihenfolge Management
