## Projektbeschreibung

**Remember Me** ist eine Progressive Web App (PWA) für Trainingsplanung und Übungsverwaltung. Die Anwendung ermöglicht es Benutzern, ihre Trainingsroutinen zu erstellen, zu verfolgen und zu optimieren. 

Die App ist vollständig responsiv und funktioniert nahtlos auf:
- 📱 Mobilgeräten (mit touch-optimierter Steuerung)
- 💻 Desktop/Tablet (mit mouse/keyboard-optimierter Steuerung)
- 📴 Offline (mit Service Workers für PWA-Funktionalität)

## Glossar

| Begriff | Beschreibung |
|---------|-------------|
| PWA | Progressive Web App - Web-App mit nativen App Features |
| Modul | Eine in sich geschlossene Komponente des Systems |
| Anforderung | Eine funktionale oder nicht-funktionale Anforderung |
| Spezifikation | Technische Details zur Implementierung |
| Service Worker | JS-Worker für Offline-Funktionalität und Caching |
| Responsiv | Design passt sich an verschiedene Bildschirmgrößen an |
| Touch-optimiert | UI-Elemente sind für Finger-Bedienung optimiert |
| Workout | Ein Trainingsplan mit Übungen |
| Übung | Ein einzelner Trainingsbewegung mit Sets/Reps |
